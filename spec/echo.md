# Echo

Echo received arguments to stdout with a trailing newline.

## Invocation

`echo message`

- `message` - All arguments given to program, separated by spaces and treated as a single string.

### Examples

| Input                | Output          |
| -------------------- | --------------- |
| `echo Hello, World!` | `Hello, World!` |

## Implementation

### Code

```
println(message)
```
